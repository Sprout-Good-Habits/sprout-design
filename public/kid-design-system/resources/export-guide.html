Component Export

Grab any Sprout component as a self-contained CSS bundle. No build system, no framework dependencies.

How it works

Each bundle includes design tokens + all CSS dependencies concatenated into a single file. Every CSS variable has an inline fallback, so bundles render correctly without any other Sprout infrastructure.

Option 1: Hotlink from Vercel

Link directly to a pre-built bundle:

<link rel="stylesheet" href="https://sprout-design.vercel.app/export/chat-messages/chat-messages.bundle.css">

Option 2: Run the export script locally

Clone the repo and run:

node scripts/export-component.mjs chat-messages --out ./my-export

This generates the CSS bundle plus any Rive assets and JS helpers the component needs.

Option 3: Copy from the manifest

Read component-manifest.json to see exactly which files a component needs, then grab them manually from /kid-design-system/components/.

Customizing tokens

Bundles use CSS custom properties with hardcoded fallbacks. To override, set the variables in your own stylesheet:

:root {
  --color-brand-500: #your-brand-color;
  --spacing-md: 12px;
  --radius-xl: 16px;
}

Rive characters

Components that display the Sprout mascot or village characters need the Rive WASM runtime and the .riv asset files (included in the export). Add:

<script src="https://unpkg.com/@rive-app/canvas@2.27.0"></script>
<script src="rive-avatars.js"></script>
