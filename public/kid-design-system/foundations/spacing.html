Usage Guidelines

Recommended spacing tokens for common UI patterns. These keep the interface consistent without guesswork.

ContextTokenValueNotes
Icon gaps var(--spacing-md) 8 px Gap between icon and adjacent text or elements
Button padding var(--spacing-lg) 12 px Internal horizontal and vertical padding
Card padding var(--spacing-xl) – var(--spacing-3xl) 16 – 24 px Scales with card size; compact cards use xl, standard use 2xl–3xl
Section gaps var(--spacing-4xl) 32 px Vertical space between major content sections
Page padding var(--spacing-6xl) 48 px Outer padding of the main content area
