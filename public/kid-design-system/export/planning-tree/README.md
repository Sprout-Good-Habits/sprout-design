# planning-tree - Sprout Design Component Bundle

## Quick start

```html
<link rel="stylesheet" href="planning-tree.bundle.css">
```

## What's included

**CSS bundle** (`planning-tree.bundle.css`):
- Design tokens (CSS custom properties)
- chat-messages.css
- planning-tree.css


**JS helpers:**
- planning-tree.js
- chat-messages.js

## Tokens

All CSS custom properties have inline fallback values, so the bundle renders
correctly as-is. To customize, override the variables in your own stylesheet:

```css
:root {
  --color-brand-500: #your-brand-color;
  --spacing-md: 12px;
}
```

## Rive characters

This component does not use Rive assets.
