Button Utility
Pill-shaped utility buttons used in toolbars for navigation, actions, stat counters, and avatar groups. They sit in the leading and trailing slots of toolbar variants.
  • ✓ Use for toolbar navigation (Back, Cancel).
  • ✓ Use for displaying gamified stat counters (fire streak, gems).
  • ✓ Use for showing avatar groups in chat headers.
  • ✓ Use for icon-only actions within toolbars (settings, profile).
  • ✗ Don’t use as a primary action button — use Buttons instead.
  • ✗ Don’t use outside of toolbars or navigation contexts.

Accessibility

  • Use <button> elements, not <div>, so they are focusable and activatable via keyboard.
  • Add aria-label to icon-only buttons (e.g., aria-label="Go back", aria-label="Close").
  • Disabled buttons should include aria-disabled="true" and tabindex="-1".
  • Stat counters (icon + value) should include aria-label describing the stat (e.g., aria-label="Fire streak: 100").
  • The pressed state must shift down visually to provide tactile feedback, matching the shadow removal.
  • Ensure minimum 44 × 44px touch target for mobile (the 12px padding achieves 48px for icon-only variants).
