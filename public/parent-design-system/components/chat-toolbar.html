Chat Toolbar

Chat header with frosted gradient overlay, toolbar pills containing navigation and avatar controls. Sits below the status bar.

CSS file
chat-toolbar.css
Last update
Tue Apr 15, 2026

Overview

The toolbar provides navigation (back button) and context (edit button, participant avatars). Uses a frosted gradient that fades content as it scrolls behind the header area.

Anatomy

  • .header-frost - Gradient overlay
  • .chat-header - Positioned container
  • .toolbar-pill - Frosted glass pill
  • .toolbar-icon-btn - 36px icon buttons
  • .chat-header-avatars - Overlapping avatar stack

Usage

Requires avatar.css for the stacked avatars. Place .header-frost and .chat-header inside .device-screen after the status bar.
