Parent Message

Outgoing message from the parent to Sprout. Blue bubble (brand-200) aligned right. Supports text, quoted replies, photo attachments, and file attachments.

CSS file
chat-messages.css
Last update
Thu Apr 17, 2026

Overview

Parent messages are right-aligned blue bubbles. They can include quoted replies (when replying to a specific message), photo thumbnails, and file attachments above the text bubble. The message group supports the swipe-to-reply gesture.

Anatomy

  • .msg-group.msg-outgoing - Container aligned right
  • .msg-row.msg-outgoing - Row with bubble (max-width 80%)
  • .msg-bubble.msg-bubble-out - Blue bubble (brand-200, radius-3xl 20px, padding spacing-lg 12px)
  • .msg-col - Column wrapper for bubble
  • .msg-bubble-mention - Optional in-bubble mention quote when replying to a specific message
  • .msg-attachments - Photo + file grid above bubble

Usage

Use the parent message for all user-sent content in the chat. The blue bubble visually distinguishes parent messages from Sprout responses.

Text only

Nice job Emma! Can we add a reminder for Noah about the bed?

With mention quote

When replying to a specific message, the quoted text appears inside the bubble as the first element.

Emma completed her morning routine streak...
That's amazing! Let's keep it going.

With photo attachments

Photos appear as 108x108 thumbnails in a grid above the bubble. Uses the Media component.

Here are the photos from today

With file attachment

File attachments render as a card with icon, filename, and type above the bubble. Uses the Media component.

Weekly report.pdf
PDF
Here's the report from school

Usage (continued)

Multiple messages in a group

Consecutive outgoing messages stack right-aligned with spacing between groups.

Hey, just dropped Emma off at school
She has her lunch and water bottle
Pick up is at 3:15 today

Spec

.msg-bubble-out with domain tags KN, CN, AC. Content variants: text, voice transcription, photo, file, quoted reply.
