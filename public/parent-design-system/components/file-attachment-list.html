Overview

The file attachment list is a pattern that combines photos and files in a single attachment grid above an outgoing message. Photos render as square thumbnails, files render as full-width cards. The grid wraps as needed and aligns to the right edge. This pattern appears when a parent sends a message with multiple attachments of mixed types.

Individual media items (images and file cards) are part of the Media component, which also provides loading states. This page documents the grid composition pattern.

Anatomy

  • .msg-attachments . Grid container (flex-wrap, gap 4px, right-aligned)
  • .msg-attachments-img . Photo thumbnail (108x108, radius 20px)
  • .msg-attachments-file . File card (full width within grid)
  • .msg-group.msg-outgoing . Parent container for the complete message

Usage

Use the file attachment list when a parent sends a message with multiple attachments of mixed types (photos and documents together).
