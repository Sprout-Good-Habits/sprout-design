Chat Context Menu

Bottom sheet system with scrim overlay, handle, and list items. Includes the generic sheet base (scrim, handle, header, close button) and the context menu variant (Copy, Reply, Forward, Delete).

Overview

The sheet is the base overlay component used by multiple features. It slides up from the bottom with a spring curve animation. The context menu variant appears on long-press of a message and offers Copy, Reply, Forward, and Delete actions. Delete uses the destructive red styling.

Anatomy

  • .sheet-scrim - Semi-transparent overlay behind the sheet
  • .sheet - Slide-up container
  • .sheet-top - Handle area at the top of the sheet
  • .sheet-handle - Drag indicator bar
  • .sheet-header - Header row with title and close button
  • .sheet-title - Title text in the header
  • .sheet-close - Close button in the header
  • .sheet-list - Generic list container
  • .sheet-list-item - Generic list item
  • .ctx-menu-list - Context menu list container
  • .ctx-menu-item - Context menu item (Copy, Reply, Forward)
  • .ctx-menu-item.destructive - Red delete variant

Usage

Place .sheet-scrim and .sheet as siblings inside .device-screen. Toggle .open on both. The context menu reuses the sheet base with .ctx-menu-list content.
