Photo Upload

Attachment selection panel for sending photos, taking camera shots, or attaching documents. Bottom panel with animated row entries.

CSS file
attach-panel.css
Last update
Thu Apr 17, 2026

Overview

The photo upload panel slides up from the composer area. It presents three options: Photos (from gallery), Camera (take photo), and Document (file picker). Each row animates in with a staggered translateY transition. The panel collapses back when an option is selected or dismissed.

Anatomy

  • .attach-panel . Collapsible container (max-height transition)
  • .attach-panel.open . Expanded state (276px)
  • .attach-panel-row . Individual option row (white card, shadow, 16px padding)
  • .attach-panel-icon . 40px circle icon with colored background
  • .attach-panel-icon--photos . Blue icon (d1e0ff)
  • .attach-panel-icon--camera . Gray icon (f5f5f5)
  • .attach-panel-icon--document . Brand icon (e0f2fe)
  • .attach-panel-label . Option label text (16px)

Usage

Show the photo upload panel when the user taps the attachment button in the chat composer. Dismiss it when an option is selected or the user taps outside.
