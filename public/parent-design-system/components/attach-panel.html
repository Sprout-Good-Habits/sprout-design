Attach Panel

Attachment input system with three variants. The inline plus-pill expands the + button into camera/image/file icons with iOS spring animations. The attach panel shows stacked rows (Photos, Camera, Document) below the composer. Also includes the decorative iOS keyboard.

Overview

Two attachment flows are supported. Flow A (inline): the + pill expands to reveal camera, image, and file buttons with the + rotating to become an X. Flow B (panel): the + button toggles a panel of stacked rows below the composer, swapping with the keyboard. Both use iOS-style spring curve animations. The keyboard is decorative and slides up/down with matching transitions.

Anatomy

  • .plus-pill - Expandable gray pill container
  • .plus-pill-plus - + icon that rotates 45 degrees to become an X
  • .plus-pill-extra - Container for camera/image/file icons
  • .attach-panel - Stacked row container for Flow B
  • .attach-panel-row - Individual option row with icon (Photos, Camera, Document)
  • .ios-keyboard - Decorative QWERTY keyboard

Usage

The plus-pill goes inside .composer-toolbar-left. The attach-panel and ios-keyboard go inside .composer-area after the composer row. Use .expanded class on .plus-pill for Flow A, .open class on .attach-panel for Flow B.
