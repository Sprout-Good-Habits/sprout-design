Confirm Card

Structured confirmation card with key-value rows and confirm/cancel buttons. Used when Sprout asks the parent to verify details before an action.

CSS file
confirm-card.css
Last update
Tue Apr 15, 2026

Overview

The card presents structured data (e.g., quest name, frequency, duration) in labeled rows with a bottom divider. Below the data, confirm and cancel buttons allow the parent to approve or reject. Buttons support done and dismissed states.

Anatomy

  • .confirm-card - Bordered card container
  • .confirm-card-title - Bold heading at the top of the card
  • .confirm-card-row - Key-value pair row
  • .confirm-card-label - Label text within a row
  • .confirm-card-value - Value text within a row
  • .confirm-card-desc - Footer description text
  • .confirm-card-btns - Button row container
  • .confirm-card-btn-confirm - Confirm button variant
  • .confirm-card-btn-cancel - Cancel button variant

Usage

Place inside a .msg-group as a block element. The card stretches to full width within its container.
