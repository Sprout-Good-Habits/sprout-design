Media

Unified component for images and file attachments in chat. Covers both the in-message attachment view and the composer preview strip, with default and loading states.

Overview

The Media component handles every visual representation of user-attached content in chat: photo thumbnails, file cards, and their composer previews. Each variant supports a loading state with a spinner overlay or icon swap.

This component consolidates styles previously split across chat-messages.css and chat-composer.css. The composition patterns (grids, lists) are documented separately in Media Gallery, File Attachment, and File Attachment List.

Anatomy

Attachment context (in-message)

  • .media-img - Image thumbnail (108x108, radius 20px)
  • .media-file - File card (flex row, white bg, border, radius 16px)
  • .media-file-icon - Icon container (36x36, brand-500 bg, white icon)
  • .media-file-info - Text column (flex 1)
  • .media-file-name - Filename (14px, semibold, truncated)
  • .media-file-meta - File type label (14px, text-tertiary)

Composer context (preview)

  • .media-thumb - Base thumbnail (120x120, radius 20px)
  • .media-thumb--image - Image preview modifier
  • .media-thumb--file - File preview card (white bg, border)
  • .media-thumb-file-label - File type text ("PDF")
  • .media-thumb-file-name - Filename (3-line clamp)
  • .media-thumb-x - Close button (36x36 circle, top-right)

Loading utilities

  • .media-spinner - 20px CSS spinner (border animation)
  • .media-spinner--lg - 24px variant for file icon replacement
  • .media-spinner--muted - Gray variant for light backgrounds
  • .media-overlay - Semi-transparent overlay (rgba(0,0,0,0.2))
  • .is-loading - State class to activate loading visuals

Loading states

Add the .is-loading class to any media element to activate its loading state:

  • Images (attachment and composer): Include a .media-overlay with a nested .media-spinner. The overlay covers the thumbnail with a dark scrim.
  • Files (attachment): Include a .media-spinner.media-spinner--lg inside .media-file-icon. The spinner replaces the SVG icon (hidden via CSS).
  • Files (composer): Include a .media-overlay with a nested .media-spinner, same as images.

Spec

  • Image attachment: 108x108, radius 20px, object-fit cover
  • Image composer: 120x120, radius 20px, object-fit cover
  • File card: width 100%, padding 12px, radius 20px, white bg, no border, overflow clip
  • File icon: 36x36, radius 4px, brand-500 bg, white 24px icon
  • File text: name 14px/20px semibold text-primary, meta 14px/20px regular text-tertiary
  • Composer thumb: 120x120, radius 20px
  • Close button: 36x36 circle, bg-tertiary, top-right inset 8px
  • Spinner: 20px (default) / 24px (lg), 2px border, 0.6s rotation
  • Overlay: rgba(0,0,0,0.2)
