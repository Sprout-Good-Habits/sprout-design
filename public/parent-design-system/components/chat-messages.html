Chat Messages

Complete message system for the chat thread. Includes bubbles (incoming, outgoing, co-parent), sender labels, actions (copy, thumbs up/down), quoted replies, swipe-to-reply, system messages, typing indicator, and attachment grids.

Overview

Messages are grouped by sender. Each group contains a sender label, optional thought toggle, one or more bubble rows, and action icons. Outgoing messages align right with blue bubbles. Incoming messages align left with white bubbles and an avatar. Co-parent messages use a sky-blue bubble.

Anatomy

  • .chat-thread - Scrollable container
  • .msg-group - Message cluster
  • .msg-row - Avatar + bubble row
  • .msg-bubble / .msg-bubble-in / .msg-bubble-out / .msg-bubble-coparent - Bubble variants
  • .msg-sender - Name label
  • .msg-actions - Copy/thumbs row
  • .msg-quoted - Reply quote
  • .msg-system - Centered timestamp
  • .typing-indicator - Thinking state
  • .msg-attachments - File/photo grid

Usage

The chat thread auto-scrolls and hides scrollbars. The thread bottom is dynamically adjusted by JS to accommodate the composer height.
