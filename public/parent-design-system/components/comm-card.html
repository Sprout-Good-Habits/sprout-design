Communication Card

Inline preview of a drafted email or message that Sprout composed on behalf of the parent. Shows the channel, recipients, subject, and full body so the parent can review before sending.

CSS file
comm-card.css
Last update
Thu Apr 17, 2026

Overview

When Sprout drafts an email or message for the parent, the communication card previews the full content inline in the chat. The parent can read the draft, then approve or edit it via the accompanying button stack.

Anatomy

  • .comm-card - Card container (330px, radius-3xl, white bg, subtle shadow)
  • .comm-card-header - Channel row: icon (mail, message) + type label + expand button
  • .comm-card-type - Channel name ("Email", "Message")
  • .comm-card-expand - Expand button to open full preview
  • .comm-card-field - Metadata row (To, Subject, Via, etc.)
  • .comm-card-field-label - Bold field name
  • .comm-card-field-value - Field content
  • .comm-card-body - Free-form message body

Usage

Always pair with a button stack below the card so the parent can approve or edit the draft. The card sits inside .msg-col as a sibling to the text bubble, followed by the button stack.

  • Primary button: "Send" or "Send email"
  • Secondary button: "Edit" or "Cancel"

Content variants

  • Email: Mail icon, To + Subject fields, full body with greeting and sign-off
  • Message: Chat icon, To field only (no subject), shorter body
