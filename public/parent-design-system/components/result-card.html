Result Card

Inline card for research results, web results, and recommendations that Sprout surfaces during a conversation. Shows a category tag, title, and description with an expand button to view the full source.

Overview

When Sprout searches the web, references research, or generates a recommendation, it surfaces the result inline as a result card. The card gives the parent a quick preview with enough context to decide whether to expand the full source.

Anatomy

  • .result-card - Card container (330px, radius-3xl, white bg, subtle shadow)
  • .result-card-header - Type row: icon + type label + expand button
  • .result-card-type - Source type ("Web result", "Research result", "Recommendation")
  • .result-card-expand - Expand button to open full content in a sheet
  • .result-card-body - Content area with category, title, and description
  • .result-card-category - Optional source callout (12px, tertiary). Keep concise: a URL, source name, or short label. Omit when the source is obvious from context.
  • .result-card-title - Result title (16px, semibold)
  • .result-card-desc - Summary text (14px, regular)
