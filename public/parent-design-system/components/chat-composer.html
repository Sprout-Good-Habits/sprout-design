Overview

The composer is the primary input surface. It adapts between default (mic icon), active (send icon when text present), recording (waveform), and transcribing (spinner) states. The plus button triggers attachment flows. The @mention button opens a participant picker.

Anatomy

  • .composer-area - Absolute positioned container
  • .composer-pill - White rounded card
  • .composer-input - Contenteditable field
  • .composer-toolbar - Icon row (plus, @mention, mic/send)
  • .btn-mic - Tri-state: mic / send / stop
  • .attach-strip - Thumbnail previews
  • .reply-pill - Quoted reply in composer
  • .voice-recording-toolbar / .voice-transcribing-row - Voice states
  • .mention-popup - Floating picker for people and projects
  • .mention-item - Row inside popup (avatar/icon + name + subtitle)
  • .mention-tag - Inline non-editable tag inserted after selection

Usage

Position inside .device-screen at the bottom. The composer area uses a ResizeObserver pattern to adjust the chat thread's bottom offset dynamically.

Default state

Empty input with placeholder text, plus button, @mention button, and mic icon.

Active state (text entered)

When text is present the mic icon switches to a send arrow.

With inline mention tag

After selecting a person or project from the popup, a non-editable .mention-tag is inserted inline. The tag uses contentEditable="false" so it behaves as an atomic unit.

@Mention popup

Triggered by typing @ in the composer or tapping the @ toolbar button. The popup floats above the composer and lists people (co-parent, Sprout AI) and projects. Typing after @ filters results by name or subtitle. Selecting a row inserts a .mention-tag and closes the popup.

Popup anatomy

  • .mention-popup - Positioned absolute, bottom: 100%, white card with radius-3xl and subtle shadow
  • .mention-item - Flex row with 8px/12px padding, hover highlight
  • .mention-item-avatar-wrap - Rive-animated avatar (.avatar.avatar-xs) for people
  • .mention-item-icon - 24x24 icon slot for projects (folder icon)
  • .mention-item-info - Name + subtitle, inline for people, stacked (.stacked) for projects
  • .mention-item-name - 14px medium weight
  • .mention-item-sub - 12px regular, text-secondary

Behavior

  • Popup opens when user types @ (preceded by space or start of input) or taps the @ toolbar button
  • Results filter in real time as the user types after @
  • Tapping a row inserts <span class="mention-tag">@Name</span> as a non-editable inline element
  • Popup closes on selection, blur (150ms delay), or when the query contains a space
  • The popup hides suggestion chips while open (adds .faded to chip container)
  • Mic button switches to send mode as soon as the @ character is entered

Member types

The mention popup supports two entity types, each with distinct visual treatment:

  • Person - Rive-animated avatar (.avatar.avatar-xs), name + role subtitle displayed inline. Uses isSprout flag to select Rive source
  • Project - Folder icon (24px), name + "Project" subtitle displayed stacked (.mention-item-info.stacked)
