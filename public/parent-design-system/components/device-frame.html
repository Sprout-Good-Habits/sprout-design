Device Frame

iPhone and iPad mockup frame for product explorer pages. Provides the outer chrome, inner shadow, screen container, and home indicator.

CSS file
device-frame.css
Last update
Tue Apr 15, 2026

Overview

The device frame wraps all screen content in a realistic mobile mockup. It supports iPhone (default 402x874) and iPad (640x854) via the .ipad modifier class. The screen background should be set per page.

Anatomy

  • .device-wrapper - Outer chrome with shadow
  • .device-inner-shadow - Inner ring with inset shadow
  • .device-mockup - Screen border and clip
  • .device-screen - Content viewport, 402x874 default
  • .home-indicator - Bottom bar

Usage

Import device-frame.css and wrap content in .device-wrapper > .device-inner-shadow > .device-mockup > .device-screen. Add .ipad class to .device-wrapper for tablet.
