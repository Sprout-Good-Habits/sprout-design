File Attachment

Inline file attachment card in a message. Shows file icon, filename, and file type label. Used inside outgoing message attachment grids.

Last update

Overview

File attachments appear as compact cards inside the message attachment grid. Each card displays a colored icon (brand blue), the filename (truncated with ellipsis if long), and the file type. Files are full-width within the attachment container.

The individual file card is part of the Media component (.media-file), which also provides loading states. This page documents the card's usage within message layouts.

Anatomy

  • .msg-attachments-file . File card (flex row, white bg, border, radius 16px, padding 12px 16px)
  • .msg-attachments-file-icon . Icon container (36x36, radius 4px, brand-500 bg, white icon)
  • .msg-attachments-file-info . Text column (flex 1)
  • .msg-attachments-file-name . Filename (14px, semibold, truncated)
  • .msg-attachments-file-meta . File type label (14px, text-tertiary)

Usage

Use the file attachment card to display non-image files (PDFs, documents, spreadsheets) within outgoing message attachment grids.
