Sprout Message

Incoming message from Sprout to the parent. White bubble with avatar, sender indicator, reasoning toggle, and message actions (copy, thumbs up/down).

Overview

The Sprout message is the primary incoming message component. It includes a Rive-animated avatar, a sender indicator with optional reasoning toggle ("Show thinking"), the message bubble in white, and action buttons for copy and feedback. Messages can optionally include inline images, quoted replies, and attached cards.

Anatomy

  • .msg-group.msg-incoming - Message container aligned left
  • .msg-row.msg-incoming - Row with avatar + bubble
  • .msg-bubble.msg-bubble-in - White bubble (radius-3xl 20px, max-w 264px, padding spacing-lg 12px)
  • .msg-avatar - Avatar wrapper (flex-end aligned)
  • .sender-indicator - Name + reasoning toggle above bubble
  • .msg-bubble-mention - Optional in-bubble mention quote when replying to a specific message
  • .msg-col - Column wrapper inside msg-row, holds bubble + optional cards and button stacks as siblings
  • .msg-actions - Copy, thumbs up, thumbs down row below (Sprout messages only, not shown on parent or co-parent messages)

Cards (connector cards, confirm cards, artifact cards) are inserted as siblings to the bubble inside .msg-col. They sit below the text bubble and can be wider than the bubble (e.g. connector cards at 330px vs bubble at 264px).

Usage

Use the Sprout message for all AI-generated responses in the parent chat. The sender indicator distinguishes Sprout from co-parent messages.

Text only

Sprout
Good morning! Emma completed her morning routine streak: 5 days running.

With mention quote

When Sprout replies to a specific parent message, the quoted text appears inside the bubble as the first element.

Sprout
Can we add a reminder for Noah?
Absolutely! I've set a bedtime reminder for Noah at 8:30 PM.

With images

Horizontal row of 108x108 thumbnails above the bubble. Uses the Media component. Can scroll when many images are present.

Sprout
Here are some activity ideas for this weekend!

With inline card

Cards (connector, confirm, artifact) are siblings to the bubble inside .msg-col, sitting below the text.

Sprout
Sure! Here is the Gmail connector.
Google calendar
Manage your schedule

With button stack

Action button pairs (primary + secondary) sit below the bubble inside .msg-col, inserted before the actions row. Buttons stretch full-width and are indented past the avatar.

Sprout
You have 2 pending habit completions from Noah. Would you like to approve them?

With file attachment

File attachments appear above the bubble as a card with icon, filename, and file type. Uses the Media component. For incoming messages, the grid is indented past the avatar.

Sprout
Emma's progress report.pdf
PDF
Here's Emma's weekly progress report. She completed 12 out of 15 habits this week!

Spec

.msg-bubble-in with domain tags KN, CN, AC, TS. Context variants: knowledge response, action confirmation, error message, attachment carrier.
