Co-parent Message

Incoming message from a co-parent. White bubble aligned left with a custom photo avatar or silhouette fallback. No reasoning toggle or action buttons (those are Sprout-only).

CSS file

Last update

Overview

Co-parent messages are incoming messages from another caregiver in the shared chat. They use the same white bubble as Sprout messages but are distinguished by: the co-parent's name in the sender indicator, a photo avatar (or silhouette if no avatar is set), and no action buttons or reasoning toggle.

Anatomy

  • .msg-group.msg-incoming - Message container aligned left
  • .msg-row.msg-incoming - Row with avatar + bubble
  • .msg-bubble.msg-bubble-coparent - White bubble (same style as .msg-bubble-in: radius-3xl 20px, max-w 264px, padding spacing-lg 12px)
  • .msg-avatar - 32px avatar: custom photo or silhouette fallback
  • .sender-indicator - Co-parent's display name (no reasoning toggle)
  • .msg-bubble-mention - Optional in-bubble mention quote when replying to a specific message

Co-parent messages do not show action buttons (copy, thumbs up/down). Those are reserved for Sprout messages only.

Usage

Use the co-parent message for all messages sent by another caregiver in the shared parent chat.

Text only

Randall
Hey, can you check if Emma finished her homework?

With mention quote

When a co-parent replies to a specific message, the quoted text appears inside the bubble.

Randall
Can you check if Emma finished her homework?
Yes, she just wrapped up! All done.

With photo attachments

Photos appear as 108x108 thumbnails above the bubble. Uses the Media component.

Randall
Here are the photos from the park today

With file attachment

File attachments render as a card with icon, filename, and type above the bubble. Uses the Media component.

Randall
Noah's schedule.pdf
PDF
Updated Noah's after-school schedule for next week

Multiple messages in a group

Consecutive messages from the same co-parent. The avatar only appears on the last message; earlier messages use a spacer to keep alignment.

Randall
Hey, running a bit late
Can you start dinner without me?

Spec

.msg-bubble-coparent shares all styles with .msg-bubble-in (white bg, padding spacing-lg 12px, radius-3xl 20px, max-w 264px). The difference is behavioral: no action buttons, no reasoning toggle, and the avatar is a Rive village character instead of the Rive-animated Sprout.
