Overview

A bottom sheet menu with actions for managing a connected API service. Contains "View details" (neutral), "Disconnect" (destructive), and "Delete" (destructive) options. Extends the existing .sheet pattern.

Anatomy

  • .connector-more-sheet - Modifier on .sheet (padding-bottom spacing)
  • .connector-more-item - Menu item button with icon + label
  • .connector-more-item.destructive - Red variant for dangerous actions

Usage

Opened by the overflow dots button on connector cards and list rows. Uses the existing .sheet + .sheet-scrim pattern for the bottom sheet behavior. Each item is a full-width button.
