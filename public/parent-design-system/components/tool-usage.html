Tool Usage

Collapsible inline indicator showing when Sprout is using a tool (e.g., "loading family overview", "checking schedule"). Displays as a single line that can expand to show details.

CSS file
tool-usage.css
Last update
Tue Apr 15, 2026

Overview

Tool usage indicators appear in the chat thread during processing. The label text cycles through activities with a fade transition. Clicking the chevron expands to show the full detail text. The chevron rotates 180 degrees when open.

Anatomy

  • .tool-usage - Clickable row container
  • .tool-usage-chevron - Expand/collapse arrow that rotates on open
  • .tool-usage-label - Activity text with fade animation
  • .tool-usage-details - Expandable detail text area

Usage

Place inside the chat thread as a standalone element. Toggle .open on .tool-usage-chevron and .tool-usage-details to expand.
