Overview

The Skill Sheet is a bottom sheet that shows the full detail of something Sprout has learned: a routine, a set of rules, a connected service's configuration. It opens when the parent taps on a skill Card in the chat thread or in skill management.

Extends the .artifact-sheet base with a centered title and subtitle header, a back button, and a scrollable body that supports prose, key-value field rows, and lists.

Anatomy

  • .skill-sheet - Modifier on .artifact-sheet (93% max-height)
  • .skill-sheet-back - Pill-shaped back button (44px, border-tertiary, shadow)
  • .skill-sheet-title - Centered container for title + subtitle
  • .skill-sheet-title-text - Skill name (text-md, semibold)
  • .skill-sheet-subtitle - Secondary info like "Last updated today" (text-xs, medium)
  • .skill-sheet-spacer - Invisible 44px spacer to balance the back button
  • .artifact-sheet-body - Scrollable content area (prose, fields, lists)
  • .skill-sheet-field - Key-value row with label and value

Content

The body is free-form and supports any combination of:

  • Prose - Paragraph description of what Sprout learned
  • Key-value fields - .skill-sheet-field rows with label and value (type, schedule, source)
  • Bulleted lists - Steps, rules, or items
  • Numbered lists - Ordered sequences

Behavior

  • Opens when the parent taps a skill Card (badge variant with Active/Paused/Consent status).
  • Back button dismisses the sheet and returns to the chat thread.
  • Scrollable body for long content. The header stays fixed.
