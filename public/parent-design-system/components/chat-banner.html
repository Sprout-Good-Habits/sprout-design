Chat Banner

Notification banners that appear at the top of the chat. Variants include reconnection (with spinner), delete undo (with undo button), and toast (generic notification). Also defines the shared rspin keyframe animation.

chat-banner.css
Last update

Overview

Banners float above the chat thread content. They appear with a fade-in animation and can auto-dismiss with a fade-out + upward slide. The reconnection variant includes an animated spinner.

Anatomy

  • .banner - Base with fade transition
  • .banner.show / .banner.hiding - Visibility states
  • .banner-icon - 20px icon
  • .banner-text - Message
  • .banner-undo - Action link
  • .banner-reconnect - Spinner variant
  • .banner-delete - Delete variant
  • .banner-toast - Toast variant

Usage

Place banners inside .device-screen. Toggle .show class to display. Add .hiding class before removing .show for the exit animation.
