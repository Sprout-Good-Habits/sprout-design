Chat Chips

Horizontally scrollable chip strips. Option chips are single-line choices from the AI (e.g., "Every night", "Weekdays only"). Suggestion chips are two-line pre-conversation prompts with title and subtitle.

CSS file
chat-chips.css
Last update

Overview

Chips provide quick-tap responses. Option chips appear after an AI message with choices. Suggestion chips appear before any conversation starts, offering conversation starters. Both scroll horizontally and hide scrollbars.

Anatomy

  • .option-chips - Scroll container
  • .option-chip - Individual chip with hover/selected/faded states
  • .suggestion-chips - Pre-convo container
  • .suggestion-chip-title - Bold first line
  • .suggestion-chip-sub - Tertiary second line

Usage

Place .option-chips or .suggestion-chips inside the chat thread or composer area. Chips can be dynamically generated. Selected chips get the .selected class; unselected ones get .faded.
