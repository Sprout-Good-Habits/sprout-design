Artifact

Inline artifact card and detail bottom sheet. Used when Sprout creates or references a resource (e.g., a bedtime routine, a habit plan). The card previews the artifact; tapping opens a full detail sheet.

Overview

Artifacts are Sprout-generated resources embedded in the chat. The inline card shows an emoji, name, version badge, and "View" link. The detail sheet slides up with title, scrollable body content, and an "Assign to child" action button.

Anatomy

  • .artifact-card - Clickable inline card container
  • .artifact-icon - Emoji representing the artifact type
  • .artifact-body - Container for name and subtitle
  • .artifact-name - Artifact title text
  • .artifact-version - Version badge
  • .artifact-sub - Subtitle or description text
  • .artifact-link - "View" link element
  • .artifact-sheet - Bottom sheet container
  • .artifact-sheet-scrim - Semi-transparent overlay behind the sheet
  • .artifact-sheet-header - Sheet header with title
  • .artifact-sheet-body - Scrollable body content area
  • .artifact-sheet-footer - Footer section
  • .artifact-sheet-assign - "Assign to child" action button

Usage

The card goes inside a .msg-group. The sheet and scrim are siblings inside .device-screen. Toggle .open class on both scrim and sheet to show/hide.
