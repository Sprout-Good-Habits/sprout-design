Status Bar

iOS-style status bar showing time, cellular signal, wifi, and battery indicators. Positioned absolutely at the top of the device screen.

CSS file
status-bar.css
Last update
Tue Apr 15, 2026

Overview

Decorative status bar that sits at the top of every screen. Uses pointer-events:none so it doesn't block interactions with content beneath it.

Anatomy

  • .status-bar - Flex container
  • .status-bar-time - Left-aligned time
  • .status-bar-icons - Right-aligned signal indicators

Usage

Place as the first child inside .device-screen. The time text and SVG icons are hardcoded to match the iOS 17 style.
