Progress Options
Four progress-bar treatments compared side-by-side. Drag the sliders to scrub progress and change the step count.
Inside the agent interaction screen, the kid needs to see how far through a quest they are. The Figma spec explores four visual approaches — each has different connotations (Stars feel rewarding, Circles feel checklist-y, Lines are minimal, Checkpoints feel journey-like). This page renders all four off the same slider so we can compare apples to apples.
