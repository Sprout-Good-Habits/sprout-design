Goal Progress System
How quests, units, agent interactions, and parent verification turn into goal progress across kid screens. Six interactive views derived from the Pencil-down — Brand spec.

What this is. Leti's master spec covers six distinct surfaces: the home goal widget, the goal lobby, the agent interaction state machine, the goal status state machine, all card variants across tiers, and the progress-bar exploration. This explorer turns each subsection into a clickable, comparable view so we can decide what to ship.
