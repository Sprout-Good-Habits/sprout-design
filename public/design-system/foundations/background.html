Background
Backgrounds set the visual stage for every screen in Sprout. A consistent sky-and-grass pattern reinforces the outdoor, nature-themed brand identity.

When to use

  • ✓ Use Main for primary app screens, auth flows, onboarding, and splash screens.
  • ✓ Use Alt when more grass area is needed — screens with less content or where the character sits higher.
  • ✗ Don’t use card-white or plain backgrounds where the branded environment is expected.

Variants

Both variants share the same sky color and grass texture. They differ in how much grass is visible.

Main
The default background for most screens. Sky-blue fills the full screen with a thin grass strip at the bottom. The Rive character sits on the grass. Scroll behavior: Content scrolls behind the grass and character, which stay pinned at the bottom. Add generous bottom padding (e.g. 360px) so all content can scroll fully above the grass/character area.
Alt
The grass area is much taller, pushing the Rive character higher on screen. Use when screen content sits above the fold and the lower half should feel grounded. Scroll behavior: The entire page scrolls together normally — grass and character scroll off-screen as the user scrolls down. No extra bottom padding needed.

Anatomy

  • Sky — Full-bleed fill using --bg-sky. Covers the entire screen.
  • Grass edge — 24px tall rounded-top shape using --radius-4xl (24px). Provides a smooth transition from sky to grass.
  • Grass fill — Solid --bg-grass block below the grass edge. Height varies by variant.
  • Rive character (optional) — Animated Sprout character positioned on or above the grass.

Accessibility

  • Contrast: Text placed over the sky must use --text-primary or darker to meet WCAG AA (contrast ratio 14.7 : 1).
  • Motion: The Rive character animation should respect prefers-reduced-motion. Pause or hide the animation when the user prefers reduced motion.
  • Color independence: Background variants should not be the only way to communicate screen context. Always pair with distinct content and navigation.
