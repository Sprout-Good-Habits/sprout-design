Padding
Internal spacing applied within components and containers. Padding defines the breathing room between content and its boundary.

Padding Scale

Sprout padding uses the same spacing token scale. Apply these tokens to any padding property for consistent internal spacing across components.

NameValueCSS VariableSize
xxs 2 px --spacing-xxs
xs 4 px --spacing-xs
sm 6 px --spacing-sm
md 8 px --spacing-md
lg 12 px --spacing-lg
xl 16 px --spacing-xl
2xl 20 px --spacing-2xl
3xl 24 px --spacing-3xl
4xl 32 px --spacing-4xl

Usage Examples

Each box below shows a fixed 40×40 px inner element surrounded by the token's padding. The gray-50 border represents the container boundary; the violet box is the content.

md — 8 px
