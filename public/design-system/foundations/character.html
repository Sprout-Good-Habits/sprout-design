Character
Sprout is the animated mascot that guides kids through the app. Built as a Rive animation with customizable appearance, Sprout brings personality and delight to every screen.

Interactive preview

The Sprout character is a Rive state machine with customizable skin color, hair style, and mouth. The animation plays idle behaviors automatically.

Placement in screen

Sprout is positioned on the grass area, centered horizontally. The vertical position changes depending on which background variant is used.

Main background
Sprout sits at the bottom of the screen with feet on the grass edge. The character occupies the lower third, leaving the sky area open for UI content above. This is the default placement for most screens.
Alt background
Sprout is positioned higher on the tall grass, making the character more prominent. Use for character-focused screens where Sprout is the main visual element, like splash or loading states.

Anatomy

  • Rive file — sprout-character.riv. Contains all body parts, expressions, and animations in a single state machine.
  • State machine — State Machine 1. Controls idle behavior, expressions, and looking direction.
  • Native artboard — 912 × 972 px. Rive handles scaling automatically via the fit mode.
  • Alignment — Horizontally centered on the screen. Vertically anchored to the grass edge.

Customization inputs

The Rive state machine exposes number inputs for personalizing Sprout's appearance.

Placement reference

These rules are screen-size independent. The character scales and positions relative to the screen width and the grass edge.

Main — vertical position
Position the canvas so Sprout’s feet and shadow sit on the grass edge. The canvas bottom should align with the top of the grass fill, overlapping the grass edge slightly. Use position: absolute; bottom: 0 on the canvas relative to the screen, then nudge it so the shadow rests on the grass.
Alt — vertical position
Position the canvas so Sprout stands on top of the tall grass. The canvas bottom should sit at approximately 45% of the screen height from the bottom, placing Sprout in the upper portion of the grass area with feet on the grass edge.

When to use

  • ✓ Show Sprout on splash screens, loading states, and the main home screen.
  • ✓ Use the Main placement for most screens — character sits on the grass at the bottom.
  • ✓ Use the Alt placement when the character needs to be more prominent or higher on screen.
  • ✗ Don’t show Sprout on content-heavy screens where the character would obstruct UI.
  • ✗ Don’t attempt to assemble the character from individual SVG parts — always use the Rive file.

Accessibility

  • Motion: Respect prefers-reduced-motion. Pause or hide the Rive animation when the user prefers reduced motion.
  • Decorative: Sprout is decorative — the canvas element does not need an ARIA label since it doesn't convey functional information.
  • Performance: Lazy-initialize the Rive instance only when the screen is visible. Pause when off-screen to save resources.
