Color
Sprout’s color system provides 10 full scales for expressive UI, plus semantic tokens for backgrounds, text, borders, and foreground elements. Always prefer semantic tokens for standard UI; use raw scales only when semantic tokens don’t apply.

Color Scales

Each scale runs from 25 (lightest) to 950 (darkest). Swatches use live var() references so they always match the token file.
