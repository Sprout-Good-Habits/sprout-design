🏆

💎

Font Setup

Add the TossFace @font-face declaration to the top of your <style> block, then apply the font family to any element displaying emoji.

@font-face { font-family: 'TossFace'; src: url('https://static.toss.im/tossface-font/TossFaceFontWeb.otf') format('opentype'); font-display: swap; } /* Apply to emoji elements */ .emoji { font-family: 'TossFace', sans-serif; font-size: 36px; line-height: 1; }

Asset Rules

Follow these rules to keep icons and assets consistent across every Sprout page.

✓Always use the TossFace webfont for emoji — never rely on system emoji rendering
✓Use the static.toss.im CDN URL — do not use jsdelivr or other mirrors
✓Reference SVG assets by path (<img src="/brand/assets/...">) — never paste inline SVG markup
✓Use the standard size scale: 24px, 36px, 48px, or 56px — match the component context
✓Export complex Figma characters as a single composite SVG — never assemble from parts
✗Do not install icon packages (Font Awesome, Material Icons, Heroicons, etc.)
✗Do not hardcode emoji at non-standard sizes — stick to the four-step scale
✗Do not inline SVG markup from /brand/assets/ — always use <img> tags
✗Do not use CSS background-image for brand assets like the grass texture
