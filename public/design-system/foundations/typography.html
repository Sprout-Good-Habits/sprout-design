Typography
The Sprout type scale, font weights, and pairing rules — built on Inter with two distinct scales for display and body text.

Display Scale

Display sizes are used for hero text, page titles, and section headers. All display sizes use letter-spacing: -0.02em.

Display 2XL
--font-size-display-2xl 72px / 90px
Display XL
--font-size-display-xl 60px / 72px
Display LG
--font-size-display-lg 48px / 60px
Display MD
--font-size-display-md 36px / 44px
Display SM
--font-size-display-sm 30px / 38px
Display XS
--font-size-display-xs 24px / 32px

Text Scale

Text sizes are used for body copy, labels, captions, and UI elements. Text sizes use letter-spacing: 0.
