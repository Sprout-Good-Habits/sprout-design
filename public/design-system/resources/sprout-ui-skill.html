Download skill file

What it does

The Sprout UI skill teaches Claude Code how to build product screens that look and feel like the real Sprout app. Instead of writing generic UI, it uses the documented design system — correct tokens, components, patterns, and accessibility rules.

  • Component CSS — Full implementations for Button, ButtonUtility, Input, Sheet, Tile, Toast, ActionPrompt, Switch, Checkbox, Radio, PasscodeInput, ProgressBar, LoadingIndicator, AvatarGroup, Toolbar, and Social Buttons
  • Background system — Main and Alt variants with correct scroll behavior, grass, and Rive character positioning
  • Token reference — Colors, typography, spacing, radius, and shadows mapped to CSS variables
  • Component lookup — Guides AI to pick the right component for the job (e.g. use Tile for selections inside Sheets, not custom chips)
  • Composition rules — What each component allows and forbids as children
  • Accessibility — Touch targets, ARIA attributes, focus management, and contrast requirements
  • Figma integration — Steps for translating Figma designs into design-system-compliant code

Setup

Place the skill file in your project’s Claude Code skills directory:

Path: .claude/skills/sprout-ui/skill.md

Once installed, invoke with /sprout-ui followed by what you want to build. For example:
/sprout-ui build a settings screen

Requirements

  • Claude Code — CLI tool from Anthropic — docs.anthropic.com
  • tokens.css — Sprout design tokens — download here
  • Rive runtime — For the Sprout character animation — unpkg.com/@rive-app/canvas
  • Inter font — Via Google Fonts — fonts.google.com/specimen/Inter
