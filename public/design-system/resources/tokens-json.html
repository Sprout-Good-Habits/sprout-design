Tokens JSON
Machine-readable token definitions for tooling, build pipelines, and programmatic access to the Sprout design system.

tokens.json

Structured JSON containing all Sprout design tokens — colors, spacing, typography, radius, and semantic mappings. Use this for code generation, linting, or syncing tokens across platforms.

Download tokens.json

Usage

Import the JSON file into your build tooling or scripts to programmatically access token values.

JavaScript / Node

import tokens from './tokens.json';

// Access a token
const brandBlue = tokens['--brand-500']; // "#0ba5ec"

Fetch at runtime

const res = await fetch('/tokens.json');
const tokens = await res.json();
