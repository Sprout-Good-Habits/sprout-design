Village Character
A complete reference for the Rive Village Character file — every input, index value, and color mapping you need to customize the character programmatically.

State machine

The Village Character is driven by a single Rive state machine.

  • State machine name — State Machine 1
  • Artboard name — Village-character
Tip: Use these exact names when initializing the Rive runtime. The state machine name is case-sensitive.

Inputs overview

The state machine exposes the following inputs for controlling the character's appearance and mouth animation.

Input NameTypeDescription
skinIDNumber (0–15)Skin color index
hairIDNumber (0–6)Hair style index
hairshadeIDNumber (0–12)Hair colour index
beardIDNumber (0–7)Beard type index
beardshadeIDNumber (0–12)Beard colour index
clothingcolourIDNumber (0–10)Clothing colour index
mouthselectorNumber (0–20)Mouth shape / viseme ID
talkingmouth toggleBooleanEnable or disable mouth movement

Mouth shapes

Set mouthselector to a value between 0–20 (21 shapes total).

ValueDescription
0Closed mouth (default / rest position)
1–20Open mouth variations mapped to viseme phonemes
Talking animation: Set talkingmouth toggle to true to enable automatic mouth movement. When enabled, the mouth animates independently. When disabled, use mouthselector to manually set specific mouth shapes (useful for lip-sync or static expressions).

Usage example

Initialize the Rive file and set character inputs programmatically.

const r = new rive.Rive({
  src: 'village-character.riv',
  canvas: document.getElementById('character-canvas'),
  autoplay: true,
  stateMachines: 'State Machine 1',
  artboard: 'Village-character',
  onLoad: () => {
    const inputs = r.stateMachineInputs('State Machine 1');
    inputs.forEach(input => {
      switch (input.name) {
        case 'skinID':            input.value = 3;     break;
        case 'hairID':            input.value = 4;     break;
        case 'hairshadeID':       input.value = 7;     break;
        case 'clothingcolourID':  input.value = 2;     break;
        case 'beardID':           input.value = 0;     break;
        case 'talkingmouth toggle': input.value = true; break;
      }
    });
  }
});
