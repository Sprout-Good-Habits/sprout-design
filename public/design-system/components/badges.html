Pill badge

Used on colored backgrounds. Color-matched to the parent using -200 background and -700 text from the same color scale.

Anatomy

  • Leading emoji (optional) — 16px TossFace emoji icon
  • Label (required) — --font-size-text-xs regular text
  • Container — Pill shape (--radius-full), --spacing-xs vertical / --spacing-sm horizontal padding

Inline badge

Plain text with no background, used on white surfaces. Positioned at the trailing edge of a row or container.

Anatomy

  • Leading emoji (optional) — 16px TossFace emoji icon
  • Label (required) — --font-size-text-xs regular in --text-secondary
  • No container or background. Uses --spacing-xs (4px) gap between emoji and text

Status badge

A small indicator that overlays the bottom-right corner of an emoji box to communicate task submission state.

Anatomy

  • Container — --bg-primary background, --radius-xs (4px), --spacing-xxs (2px) padding on all sides
  • Icon — 12px TossFace emoji, centered
  • Positioned absolute, offset -3px from the bottom-right of the parent emoji box
Reviewing 🔄
Rejected ❎
Claimable ✅
  • Only appears in Rejected, Submitted, and Claimable states
  • Not shown in Not started or Claimed states

Content guidelines

  • Badge labels should be short: a number or a number + unit ("3 days", "10")
  • Always use TossFace emoji for leading icons, never system emoji
  • Pill badge colors must match the parent task card color variant
  • Streak badges always show a count, even when 0 (with a muted flame)

Accessibility

  • Contrast: Pill badge text uses -700 on -200 backgrounds, meeting WCAG AA contrast requirements
  • Inline badges: Use --text-secondary on white, which meets AA contrast
  • Status badges: Convey state visually via emoji. Always pair with descriptive text nearby (e.g., "Reviewing..." in the task description) for screen readers
  • Streak muting: The grayscale filter on inactive streaks is a visual cue only. The numeric label ("0 days") communicates the same information accessibly
