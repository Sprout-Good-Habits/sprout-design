Loading Indicator
Loading indicators tell users that content or an action is being processed. They reduce perceived wait time and reassure that the app is working. Sprout uses two variants: full-screen and inline.
  • ✓ Use full-screen during app launch, screen transitions, or when the entire view is loading.
  • ✓ Use inline inside buttons, cards, or sections that are loading independently.
  • ✗ Don’t use a loading indicator for actions that complete in under 300ms — it will feel like a flash.
  • ✗ Don’t use multiple full-screen indicators — only one per screen transition.

Hierarchy

Primary uses the brand blue arc (--fg-brand-primary) on a --border-primary ring. Use this as the default.

Muted uses a white arc on a white 50% opacity ring. Use this on colored or dark backgrounds where the spinner needs to stay subtle.

Size

Large (28px) for standalone loading indicators and full-screen loading.

Medium (24px) for use inside cards, sections, and inline loading states.

Small (20px) for use inside buttons and compact components.

Anatomy

  • Ring (required) — A circular track. Primary uses --border-primary (#d5d7da), muted uses white at 50% opacity.
  • Arc (required) — A quarter-circle segment that rotates continuously. Primary uses --fg-brand-primary (#0ba5ec), muted uses white (#ffffff).
  • Text (optional) — Regular text-lg in --text-secondary. Centered below the spinner with 16px gap.

Content guidelines

  • Keep text short: “Loading...”, “Saving...”, “Checking...”
  • Use an ellipsis (...) to suggest ongoing progress
  • Don’t use vague text like “Please wait” — describe the action
  • Omit text inside buttons — the spinner alone is sufficient

Accessibility

  • Role: Use role="status" on the loading container with aria-live="polite" so screen readers announce the loading state.
  • Label: If there is no visible text, add aria-label="Loading" to the spinner element.
  • Buttons: When a button enters a loading state, set aria-busy="true" and aria-disabled="true".
  • Motion: The spinning animation should respect prefers-reduced-motion. Replace with a static icon or pulsing opacity for users who prefer reduced motion.
  • Timing: If loading takes more than 10 seconds, provide a progress update or fallback message.
